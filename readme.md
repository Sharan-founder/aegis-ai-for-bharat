## Overview
AEGIS is a voice-first, multilingual civic reporting and resolution intelligence platform designed to improve how citizens interact with public systems.

## Problem
Citizens face barriers when reporting civic issues due to language limitations, misrouting, lack of transparency, and unstructured complaint data.

## Solution
AEGIS transforms unstructured citizen inputs (voice, text, images) into structured, actionable intelligence for public authorities using AI-powered classification, routing, and prioritization.

## Key Features
- Voice-based reporting
- Multilingual support
- AI classification & summarization
- Automatic department routing
- Priority scoring
- Risk hotspot detection
- Transparency tracking

- Amazon DynamoDB
- Amazon S3
- API Gateway
- Amazon QuickSight

## Impact
Improves civic accessibility, accountability, and governance efficiency at scale.
